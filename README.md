# Bird2Doge

![Bird2Doge](title.png)

WOW! Replace twitter bird with Doge

## How to install
### Chrome

1. Download this repo
2. Open `chrome://extensions`
3. Select "Load Unpackaged Extensions" and select the `src` in the DL folder
4. Open `twitter.com`❗❗❗
