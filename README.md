# Bird2Doge

![Bird2Doge](Doges/kusodasa.png)

Replace twitter bird with Doge

## How to install
### Chrome

1. Download this repo
2. Open `chrome://extensions`
3. Select `Load unpackaged extensions` and choose the folder where the DLs are located
4. Open `twitter.com`❗❗❗
